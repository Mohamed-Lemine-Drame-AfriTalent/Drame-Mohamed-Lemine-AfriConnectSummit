# AfriConnectSummit

Auteur : Mohamed Lemine Drame
Promotion : L1 Web - ISI 
 # Description du projet 

projet AfriConnect Summit 2026 est un site vitrine moderne dédié à un sommet africain 
consacré à l'innovation,au numérique et à l'entrepreneuriat.

Le site permet aux visiteurs de découvrir le programme de l'événement, 
les intervenants, les partenaires et de s'inscrire en ligne 
via un formulaire de contact.

## Fonctionnalités JavaScript implémentées

-Mode sombre / Mode clair avec sauvegarde dans le LocalStorage.
-Menu hamburger responsive.
-Navbar dynamique au défilement.
-Bouton "Retour en haut".
-Compte à rebours en temps réel jusqu'à la conférence.
-Compteurs animés au scroll.
-Animations au scroll avec IntersectionObserver (fade-in, slide-in, zoom-in).
-Onglets interactifs pour les 3 journées du programme.
-Filtrage dynamique des intervenants.
-Validation complète du formulaire avec messages d'erreur.
-Affichage automatique de l'année dans le footer.
